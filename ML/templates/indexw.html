How It Works

This system uses a machine learning model (Random Forest) trained on historical data to predict whether the irrigation pump should be activated based on current environmental conditions.

The model considers three key factors:

  • Soil Moisture: How wet or dry the soil is
  • Temperature: Current air temperature
  • Air Humidity: Humidity percentage in the air
